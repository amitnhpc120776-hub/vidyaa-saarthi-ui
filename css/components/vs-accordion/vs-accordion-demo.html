vs-accordion Component Demo

Collapsible VIS accordion with default, flush, shadow, and always-open variants.

Default (single-open)

This default accordion only keeps one panel open at a time.

Toggle between panels to see single-open behavior.

Flush (borderless, dense)

Flush styling removes container chrome for inline groupings.

Multiple panels remain open with the always-open modifier.

Shadow (elevated card)

Use shadow variant where the accordion should feel card-like.
