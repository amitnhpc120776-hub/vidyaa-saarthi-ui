• Use vs-btn-group with the vs-btn-group--horizontal (default) or vs-btn-group--vertical modifiers to control direction.
  • Apply vs-btn-group--segmented when adjacent buttons should share borders for a connected control strip.
  • Wrap multiple groups inside vs-btn-group--toolbar to create complex toolbars that wrap on smaller viewports.
  • Place any vs-btn variant or size within vs-btn-group__item; disabled buttons remain disabled inside groups.
