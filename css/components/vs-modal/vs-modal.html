Immersive workspace

Use when the task benefits from maximized space.

Content stretches to available space while retaining responsive padding. Combine vs-modal--fullscreen-responsive with layout utilities to build editors or previews.

  • Add data-vs-modal-trigger="modal-id" to any button or link to open a modal with the matching id.
  • Use data-vs-modal-dismiss on close buttons or overlay to close the modal. vs-modal--backdrop-static ignores overlay clicks.
  • Apply vs-modal--no-backdrop to remove the dimming layer; omit the overlay element for frameless layouts.
  • vs-modal--no-escape disables ESC-key closing; vs-modal--scrollable constrains body height and enables internal scrolling.
  • vs-modal--fullscreen-responsive becomes fullscreen below the medium breakpoint while honoring width constraints on larger screens.
