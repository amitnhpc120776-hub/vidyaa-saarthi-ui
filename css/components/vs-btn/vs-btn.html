• Use type="button" or type="submit" based on context; anchor tags may use role="button" with href.
  • Apply disabled (or .is-disabled) to prevent interaction. Loading state uses .is-loading and should also disable the button.
  • vs-btn--icon-only expects an accessible aria-label describing the action.
  • data-vs-toggle="true" marks a toggle button; JavaScript will sync aria-pressed and .is-active.
  • vs-btn--block stretches the button to 100% width of its container.
