Mixed Content

Showcases image with descriptive text and CTA-style controls.

Media Focus

Use vs-carousel--media to emphasize imagery.

Looping Autoplay

Combines vs-carousel--auto with vs-carousel--loop. Hover pauses.

Text-Only Fade

Use vs-carousel--text to hide media and center copy.

Manual Vertical

Applies vs-carousel--vertical with indicator-only navigation.

  • Wrap slides inside vs-carousel__track and mark the active slide with .is-active; controls and indicators are optional via modifiers.
  • Use data-vs-interval to customize autoplay speed (in milliseconds); vs-carousel--auto enables autoplay and vs-carousel--pause-on-hover pauses on hover.
  • Combine vs-carousel--slide or vs-carousel--fade with orientation modifiers (vs-carousel--horizontal / vs-carousel--vertical).
  • Apply content modifiers: vs-carousel--media for media-heavy, vs-carousel--text for text-only, vs-carousel--mixed for both.
  • Add vs-carousel--loop to allow infinite cycling; omit for edge-stops on first/last slide.
