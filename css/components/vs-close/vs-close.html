• Default shape is vs-close--square; apply vs-close--rounded or vs-close--circle to adjust curvature.
  • Use vs-close--text for the × glyph or vs-close--icon to render an SVG icon inside .vs-close__icon.
  • Combine color and style modifiers to communicate hierarchy: --default keeps backgrounds transparent, --subtle adds a light surface, --solid uses filled color, and --bordered outlines the control.
  • Size modifiers (--sm, --md, --lg) set the button dimensions and font size.
  • Apply vs-close--dismiss-parent to remove the nearest parent (or the element matching data-vs-close-target); add vs-close--no-dismiss to disable dismissal.
  • Set aria-label to describe the target being dismissed (e.g., “Close alert”); if omitted, JavaScript will default to “Close”.
